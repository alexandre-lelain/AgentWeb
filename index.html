Agent Web

Description :

Agent Web est un plugin opensource écrit en Javascript déstiné aux développeurs souhaitant intégrer un assistant personnel à leur site web.


L'agent possède quelques intéractions avec l'utilisateur : il réagit si on lui clique dessus, si on le déplace à l'aide d'un 'drag', et peut répondre à quelques phrases basiques telles que 'comment vas-tu ?', 'hello','help',...


Remarque :

Ce projet est un 'poc (proof of concept)', et n'en est qu'au stade de prototype, c'est pour cela que le nombre d'intéractions de l'agent est très limité, l'objectif étant juste de poser l'architecture (le squelette) de l'assistant web.
